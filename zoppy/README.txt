Utilizei VS Code
Para rodar o back
instalei o node
ativei os scripts no powershell
dei npm install no terminal da pasta raiz da api
iniciei com npm start

Para rodar o front
instalei o angular com npm install -g @angular/cli
depois utilizei o npm install
iniciei com ng serve
